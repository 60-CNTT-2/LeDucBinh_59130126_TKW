What is AJAX?

As defined in Wikipedia

" AJAX is a group of interrelated web development techniques used to cerate intreractive web application
or rich internet application. with AJAX, web applications can retrive data from the server
asynchronously in the background without interfering with the display and behavior of the existing page.

Brief History

Year Event
1995 Java applets were introduced - the first way to load data from client side asynchronously
1996 IFrame element were introduced, which also have this functionaly
1999 Microsoft created the XMLHTTP ActiveX control in Internet Explorer 5 which then followed by other browser as XMLHttpRequest Object
2005 The tem "Ajax" was bomed
2006 The W3C, in order to to make it official web standard, publicized the first draft spacification for XMLHttpRequest Object
